3 设计原则

  • cyan spin 锚点 — 主 spinner 保持 cyan，作为 working 行唯一纯色锚。
  • cyber silver verb — 动作词用主题 fgDim → fgMuted / fg → cyanBright glint → fgDim 的金属扫光，科技感强但不新增额外银色 palette。
  • 冷色邻近体系 — tool 蓝、tool spinner 深 teal；turn 退到 dim，避免额外跳色。
  • token 灰白化 — ↑45k ↓1.5k 不使用 teal/green；只用 dim/muted 区分非活跃/活跃。
  • 方向靠符号 — 上行/下行靠 ↑/↓ 与固定顺序区分，不靠 token 色相。
  • tps 不动 — 速度分级是用户已有肌肉记忆，保持 success/cyan/warning/error 四档。
  • idle summary 降噪 — done 行只保留耗时、token、tps、turn；删除工具成功/失败计数。
  • 对低视觉负担留底 — separator ·、tool dur、esc hint、非活跃 token 全部 dim。
